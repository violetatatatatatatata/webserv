Webserv is a C++98 project that implements a non-blocking HTTP/1.1 server.
It uses a multiplexer like 'select' or 'poll' to handle multiple clients simultaneously,
serving static files, managing uploads/deletions, and executing CGI scripts.
The server is fully configurable via an Nginx-style file, allowing for
custom ports, routes, and error pages.
