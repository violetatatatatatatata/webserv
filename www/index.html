Project Info:
Webserv is a C++98 project that implements a non-blocking HTTP/1.1 server. It uses a multiplexer (select/poll) to handle multiple clients simultaneously.

Features: serving static files, POST/DELETE methods, CGI scripts. Configurable via Nginx-style file.
